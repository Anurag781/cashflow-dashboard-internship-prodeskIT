# 💰 Cash Flow Dashboard

A simple and interactive **Cash Flow Dashboard** web application that helps users manage their **salary, expenses, and remaining balance** in a clean and user-friendly interface.

This project was developed as part of my **internship task** and demonstrates the use of **HTML, CSS, and JavaScript** to build a responsive financial tracking dashboard.

---

## 🚀 Features

* Add and manage **salary**
* Track **daily expenses**
* Automatically **calculate remaining balance**
* Simple and **clean dashboard UI**
* Dynamic updates using **JavaScript**
* Beginner-friendly financial management system

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Logic and dynamic functionality
